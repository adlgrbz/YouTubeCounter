# YouTubeCounter
YouTube Counter with Arduino and Python

## Getting Started
Download PySerial module for Python.
```python
pip install pyserial
```

Run counter.py
```python
python counter.py
```

Enter port name.
```python
Port Name (Sample: COM5): 
```
